# :mag_right: Using
___
**1. Rename the json file "data_template" to "data"** :pencil:
**2. Create ".env" file and put in: `BOT_TOKEN=your bot token`** :lock:
**3. If you need, create venv (recommended) and install libraries in the terminal:** :floppy_disk:
```bash
pip install pyTelegramBotAPI
pip install pyTelegramBotAPI requests
pip install flask
pip install python-dotenv
```
**4. Create 2 terminals: launch api.py in one, and bot.py in the other** :computer:
**5. Go to your bot and enjoy** :watermelon:
