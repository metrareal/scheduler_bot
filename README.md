# :mag_right: Using
___
:pencil: **1. Rename the json file "data_template" to "data"**
:lock: **2. Create ".env" file and put in: `BOT_TOKEN=your bot token`**
:floppy_disk: **3. If you need, create venv (recommended) and install libraries in the terminal:**
```bash
pip install pyTelegramBotAPI
pip install pyTelegramBotAPI requests
pip install flask
pip install python-dotenv
```
:computer: **4. Create 2 terminals: launch api.py in one, and bot.py in the other**
:watermelon: **5. Go to your bot and enjoy**
